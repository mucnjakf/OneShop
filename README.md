# OneShop

E-Commerce Web Application - 2021.

---

### **Overview**
OneShop is an e-commerce web application for users and merchants to buy and sell products.

---

### **Tech Stack**
- Java 11 EE
- Jakarta Servlet 5
- Hibernate 5.5.3
- MSSQL Java Database Connectivity 11
- Microsoft SQL Server
- PayPal REST API
- GlassFish 6.2.2
- Maven
- JSP - JSTL & HTML - CSS

---

### **Tools**
- Source Control: GitHub
- IDE: IntelliJ IDEA
